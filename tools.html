PhotoMap

Open Web App

About

PhotoMap is a web application that helps organise inspection photos, add notes, and display photo locations on an interactive map.

It makes it easier to keep photos from different inspections, add descriptions and locations, and share them with clients or colleagues directly from the inspection site.

Features

Upload photos from your phone or laptop.

Display photos on an interactive map.

Create separate collections for each inspection.

Add descriptions to every photo where needed.

Share inspections using a private read-only link.

Download resized photos as a ZIP archive.

Draught Survey Web App

Open Web App

About

This web application is designed for ship officers and marine surveyors to perform accurate draught survey calculations.

It allows you to keep your data securely in the cloud storage and perform surveys across multiple devices.

Features

Create and manage draught survey reports.

Save vessel information for future inspections.

Reuse hydrostatic data from previous surveys.

Add notes during the inspection.

Export results to Microsoft Excel.

Trucks Weighing Web App

Open Web App

About

Trucks Weighing Web App is a web application for cargo surveyors and tally teams that helps control cargo weight during truck operations at terminals - in ports, warehouses, and silos (loading or discharging).

Your data is stored securely in the database and can be accessed from multiple devices for team workflows.

Features

Multi-user workflow.

Track every truck during loading or discharge.

Shared notes.

Plate number autocomplete.

PDF reports.

Company logo on PDF reports.
